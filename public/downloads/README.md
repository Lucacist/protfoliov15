# Dossier des téléchargements

Placez ici les archives et logiciels téléchargeables (ZIP, EXE, DMG, etc.).

## Utilisation

1. Ajoutez vos fichiers dans ce dossier
2. Référencez-les dans vos projets avec le chemin `/downloads/nom-du-fichier.ext`

## Exemple

Si vous placez un fichier `logiciel-v1.0.zip` ici, utilisez :

```json
{
  "downloadUrl": "/downloads/logiciel-v1.0.zip",
  "downloadLabel": "Télécharger v1.0 (ZIP)"
}
```
